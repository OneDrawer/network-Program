一、光盘内容

    本光盘是《Linux网络编程》一书的配书光盘，内容包括了书中所有例题的源程序。

    本光盘根目录下有17个文件夹，其内容和含义说明如下：

    （1） 17个文件夹名对应于书中的章名，各章例题中的源程序以及所需资源放在对应的文件夹中。

    （2） 所有的源程序名和书中的源程序名称一一对应，便于读者阅读，可参阅光盘范例列表文档。



二、运行环境

    所有的源程序（除书中特别注明的错误例子外）在以下环境调试通过：

    （1）操作系统：Debian Linux 4
    （2）编辑环境：vim
    （3）编译和运行环境：GCC 4.1以上版本、GDB
    

     关于这些工具的使用方法请阅读第2章。

三、注意事项

    （1） 上机运行程序前请仔细阅读书中第2章，如果要运行Linux内核程序，请阅读第17章。

    （2） 请确保Debian操作系统的安装完整、环境变量配置正确，请仔细阅读书中的相关章节。

    （3） 在使用本光盘的程序时，请将程序复制到硬盘上，并去除文件的只读属性，注意不得改变文件名称的大小写。

    （4） 第16、17章中的程序需要在内核中运行，请先保存好相关工作的记录，防止发生内核崩溃等事件的发生造成损失。
    
    （5） 第18、19、20章中的所有程序以及资源均要按照光盘的目录结构安排，不要随意改动目录结构，否则将可能无法正常运行。
